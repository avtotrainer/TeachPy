рЃЏрЃќрЃљрЃЊ рЃЋрЃљрЃа Рђћ рЃљрЃ«рЃџрЃљ рЃЏрЃўрЃўрЃдрЃћрЃЉ **рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 05**РђЊрЃА,
рЃАрЃљрЃЋрЃАрЃћ, рЃЏрЃЊрЃўрЃЊрЃљрЃа, рЃљрЃБрЃЊрЃўрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃАрЃбрЃўрЃџрЃўрЃА рЃ░рЃћрЃюрЃЊрЃљрЃБрЃбрЃА,

­ЪЉЅ MarkdownРђЊрЃў **рЃАрЃарЃБрЃџрЃљрЃЊ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ**,
­ЪЉЅ рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃЊрЃў рЃЕрЃљрЃАрЃЏрЃБрЃџрЃўрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ,

# ­ЪЊў **рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 05 Рђћ рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ & JSON (рЃАрЃљрЃгрЃДрЃЮрЃЉрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ)**

*(рЃњрЃарЃФрЃћрЃџрЃў, рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў, рЃюрЃљрЃарЃљрЃбрЃўрЃЋрЃБрЃџрЃў рЃ░рЃћрЃюрЃЊрЃљрЃБрЃбрЃў)*

````md
# рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 05 Рђћ рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃЊрЃљ JSON

# 1. рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃБрЃюрЃЊрЃљ рЃерЃћрЃњрЃЋрЃћрЃФрЃџрЃЮрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ?

рЃљрЃЦрЃљрЃЏрЃЊрЃћ Python-рЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ РђърЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃўРђю рЃарЃљрЃдрЃљрЃфрЃћрЃЉрЃА рЃЋрЃљрЃњрЃћрЃЉрЃўрЃюрЃћрЃЉрЃЊрЃўрЃЌ:
- рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃерЃў
- рЃАрЃўрЃћрЃЉрЃерЃў
- рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃћрЃЉрЃерЃў

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЌрЃБ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃЏ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃЮрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ **рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃњрЃљрЃЌрЃўрЃерЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃљрЃф**,  
рЃЏрЃљрЃерЃўрЃю рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ **рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**.

рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА:
- рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃў  
- рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў  
- рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃћрЃЉрЃў  
- рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў  
- рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃърЃарЃЮрЃњрЃарЃћрЃАрЃў  

рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏрЃўрЃбрЃЮрЃЏ, рЃДрЃЋрЃћрЃџрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА.

# 2. рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ (РђъopenРђю) Рђћ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў

Python-рЃерЃў рЃцрЃљрЃўрЃџрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃарЃўрЃА:

```python
open(рЃцрЃљрЃўрЃџрЃўрЃА_рЃАрЃљрЃ«рЃћрЃџрЃў, рЃарЃћрЃЪрЃўрЃЏрЃў)
````

рЃарЃћрЃЪрЃўрЃЏрЃћрЃЉрЃў рЃљрЃарЃўрЃА:

| рЃарЃћрЃЪрЃўрЃЏрЃў | рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ               |
| ------ | ------------------------- |
| `"r"`  | рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ                  |
| `"w"`  | рЃЕрЃљрЃгрЃћрЃарЃљ (рЃФрЃЋрЃћрЃџрЃА рЃњрЃљрЃЊрЃљрЃцрЃљрЃарЃљрЃЋрЃА) |
| `"a"`  | рЃЏрЃўрЃгрЃћрЃарЃљ рЃЉрЃЮрЃџрЃЮрЃерЃў             |
| `"r+"` | рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ + рЃЕрЃљрЃгрЃћрЃарЃљ         |

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```python
file = open("data.txt", "w")
file.write("Hello file!")
file.close()
```

рЃћрЃА рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА **data.txt** рЃЊрЃљ рЃгрЃћрЃарЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃА.

Рџа рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃАрЃћрЃЌ рЃЎрЃЮрЃЊрЃА рЃћрЃарЃЌрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ рЃљрЃЦрЃЋрЃА:
рЃЌрЃБ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃЋрЃљрЃарЃўрЃБрЃџрЃљрЃЊ рЃерЃћрЃгрЃДрЃЊрЃћрЃЉрЃљ, рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃа рЃЊрЃљрЃўрЃ«рЃБрЃарЃЮрЃА.

# 3. рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃњрЃќрЃљ Рђћ Рђъwith open(...)Рђю рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃў

рЃћрЃА рЃљрЃарЃўрЃА **рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЮрЃЦрЃарЃЮрЃА рЃгрЃћрЃАрЃў**:

```python
with open("data.txt", "w") as f:
    f.write("Hello!")
```

рЃарЃљрЃбрЃЮрЃЏ рЃ»рЃЮрЃЉрЃўрЃљ?

* Python рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃ«рЃБрЃарЃљрЃЋрЃА рЃцрЃљрЃўрЃџрЃА
* рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃљ
* рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃўрЃф рЃЎрЃў рЃцрЃљрЃўрЃџрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃћрЃЉрЃљ

рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ **рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА**.

# 4. рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ/рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ Рђћ рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў

## рЃцрЃљрЃўрЃџрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ (write):

```python
with open("notes.txt", "w") as f:
    f.write("рЃћрЃА рЃљрЃарЃўрЃА рЃЕрЃћрЃЏрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў.\n")
    f.write("рЃЏрЃћрЃЮрЃарЃћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў.")
```

## рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ (read):

```python
with open("notes.txt", "r") as f:
    content = f.read()
    print(content)
```

## рЃ«рЃљрЃќ-рЃ«рЃљрЃќрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ:

```python
with open("notes.txt", "r") as f:
    for line in f:
        print("рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў:", line.strip())
```

# 5. рЃарЃљ рЃљрЃарЃўрЃА JSON?

JSON = JavaScript Object Notation
рЃћрЃА рЃљрЃарЃўрЃА рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:

* рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
* рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЕрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
* HTTP APIРђЊрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
* рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃўрЃА progress files
* рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

JSON рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃўрЃљ **рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ.

### JSONРђЊрЃерЃў рЃљрЃарЃўрЃА 4 рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ:

| JSON   | Python    |
| ------ | --------- |
| object | dict      |
| array  | list      |
| string | str       |
| number | int/float |

# 6. JSON рЃцрЃљрЃўрЃџрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ Python-рЃерЃў

Python-рЃА рЃљрЃЦрЃЋрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЊрЃБрЃџрЃў:

```python
import json
```

### рЃЕрЃљрЃгрЃћрЃарЃљ (json.dump):

```python
import json

data = {
    "name": "Nika",
    "age": 21,
    "skills": ["python", "linux", "git"]
}

with open("user.json", "w") as f:
    json.dump(data, f, indent=4, ensure_ascii=False)
```

# 8. рЃарЃћрЃљрЃџрЃБрЃарЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ Рђћ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉ рЃерЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў?

* рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃарЃљрЃБрЃюрЃЊрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃў
