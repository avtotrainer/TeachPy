рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЊрЃљ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃЌрЃћрЃЏрЃљрЃљ:
**Debugging, try/except, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў, рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃерЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃЌрЃљрЃю.**

­ЪЉЅ рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА, рЃўрЃАрЃћ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃБрЃџрЃў рЃЊрЃЮрЃюрЃўрЃЌрЃљрЃф.

````md
# рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 06 Рђћ Try/Except, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ Debugging

# 1. рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў Programming-рЃўрЃА рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў рЃюрЃљрЃгрЃўрЃџрЃў?

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃљрЃа рЃюрЃўрЃерЃюрЃљрЃЋрЃА РђърЃЏрЃћ рЃфрЃБрЃЊрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃў рЃЋрЃљрЃаРђю.  
рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

> РђърЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃарЃљрЃдрЃљрЃфрЃљрЃА рЃЋрЃћрЃа рЃњрЃљрЃБрЃЏрЃЎрЃџрЃљрЃЋрЃЊрЃљ, рЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃљрЃа рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЮрЃЊрЃљ рЃЏрЃљрЃА, рЃарЃљрЃф рЃЋрЃћрЃџрЃЮрЃЊрЃўрЃЌ.Рђю

рЃљрЃЏ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃерЃў рЃЋрЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃЌ:
- рЃарЃЮрЃњрЃЮрЃа рЃљрЃЏрЃЮрЃЋрЃўрЃфрЃюрЃЮрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў  
- рЃарЃЮрЃњрЃЮрЃа рЃЏрЃўрЃЋрЃЏрЃљрЃарЃЌрЃЮрЃЌ рЃЏрЃљрЃЌ try/exceptРђЊрЃўрЃЌ  
- рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃЊрЃћрЃЉрЃљрЃњрЃЮрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ  
- рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃўрЃфрЃЋрЃљрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ inputРђЊрЃћрЃЉрЃўрЃАрЃњрЃљрЃю  
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃерЃћрЃюрЃА 21РђЊрЃЎрЃљрЃарЃбрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў  

рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃљрЃарЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃБрЃюрЃљрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃ«рЃљрЃџрЃЉрЃћрЃЊрЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃљрЃЊ.

# 2. рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў Рђћ рЃарЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃА Python-рЃерЃў?

Python-рЃерЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА рЃћрЃгрЃЮрЃЊрЃћрЃЉрЃљ **Exception**.

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃћрЃЉрЃў:

| рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ | рЃарЃЮрЃЊрЃўрЃА рЃ«рЃЊрЃћрЃЉрЃљ |
|---------|------------|
| ValueError | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃбрЃўрЃърЃўрЃАрЃљрЃљ |
| TypeError | рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ ОъОАОЋОњрЃќрЃћ |
| IndexError | рЃАрЃўрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА |
| KeyError | рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ |
| FileNotFoundError | рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА |
| ZeroDivisionError | рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ |
| JSONDecodeError | JSON рЃцрЃљрЃўрЃџрЃў рЃЊрЃљрЃќрЃўрЃљрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ |

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```python
int("abc")   # ValueError
[1,2,3][5]   # IndexError
{"age": 20}["name"]   # KeyError
````

# 3. Try/Except Рђћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў

Try/except рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ PythonРђЊрЃА:

> РђърЃАрЃфрЃљрЃЊрЃћ рЃћрЃА рЃЎрЃЮрЃЊрЃў.
> рЃЌрЃБ рЃарЃљрЃЏрЃћ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ рЃгрЃљрЃЋрЃўрЃЊрЃљ, рЃљрЃа рЃњрЃљрЃљрЃЕрЃћрЃарЃЮ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Рђћ рЃЏрЃћ рЃњрЃљрЃЋрЃЏрЃљрЃарЃЌрЃљрЃЋ.Рђю

рЃцрЃЮрЃарЃЏрЃљ:

```python
try:
    risky_code()
except:
    handle_error()
```

# 4. Try/Except РђЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

```python
try:
    age = int(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃљрЃАрЃљрЃЎрЃў: "))
    print("рЃерЃћрЃюрЃў рЃљрЃАрЃљрЃЎрЃўрЃљ:", age)
except:
    print("рЃњрЃЌрЃ«рЃЮрЃЋ, рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃарЃўрЃфрЃ«рЃЋрЃў!")
```

рЃЌрЃБ student рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃАрЃљрЃА рЃљрЃЎрЃарЃўрЃЉрЃљрЃЋрЃА `"abc"` Рђћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃа рЃЕрЃљрЃЏрЃЮрЃўрЃерЃџрЃћрЃЉрЃљ.

# 5. рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ (рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ)

рЃАрЃ»рЃЮрЃЉрЃА рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃбрЃўрЃърЃў:

```python
try:
    num = int(input("Number: "))
except ValueError:
    print("рЃћрЃА рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃа рЃљрЃарЃўрЃА!")
```

рЃћрЃА рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃБрЃџрЃў рЃњрЃќрЃљрЃљ.

```python
except ValueError:
    print("рЃцрЃљрЃўрЃџрЃерЃў рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃа рЃгрЃћрЃарЃўрЃљ!")
```

# 7. finally Рђћ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ

рЃћрЃА рЃЉрЃџрЃЮрЃЎрЃў рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА Рєњ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ.

# 8. raise Рђћ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃЋрЃљ

# 9. Debugging Рђћ рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃў рЃБрЃюрЃљрЃарЃў

Debugging рЃљрЃарЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў, рЃарЃЮрЃфрЃљ рЃЋрЃўрЃЎрЃЋрЃџрЃћрЃЋрЃЌ:

* рЃАрЃљрЃЊ рЃњрЃљрЃЕрЃюрЃЊрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ?
* рЃарЃљрЃбрЃЮрЃЏ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЎрЃЮрЃЊрЃў?

# 10. Debugging рЃерЃћрЃюрЃА 21РђЊрЃЎрЃљрЃарЃбрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў

рЃерЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў (cart21.py) рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ inputРђЊрЃќрЃћ:

```python
c = input("WHICH COLUMN IS YOUR NUMBER IN? (1/2/3): ").strip()
```

рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА `"x"` Рђћ
`c in ["1","2","3"]` рЃЊрЃљрЃњрЃЋрЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃАрЃњрЃљрЃю.

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃЮрЃЊрЃўрЃЌ рЃгрЃљрЃарЃЏрЃЮрЃЋрЃўрЃЊрЃњрЃўрЃюрЃЮрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ:

РЮї рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА `""` (рЃАрЃЋрЃћрЃџрЃў Enter),
`c.strip()` рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃЊрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃерЃћрЃгрЃДрЃЋрЃћрЃбрЃЊрЃћрЃА рЃАрЃ«рЃЋрЃљ рЃюрЃљрЃгрЃўрЃџрЃерЃў.

рЃљрЃЏрЃўрЃбрЃЮрЃЏ debugger-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ:

```python
print("DEBUG: user entered:", repr(c))
```

рЃћрЃА рЃЊрЃљрЃњрЃљрЃюрЃљрЃ«рЃћрЃЉрЃА:

* рЃАрЃўрЃљ рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃфрЃљрЃарЃўрЃћрЃџрЃў
* рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃљрЃАрЃерЃў spaces
* рЃарЃЮрЃњрЃЮрЃа рЃўрЃЦрЃфрЃћрЃЋрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў

# 11. рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў try/except рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ

рЃЋрЃЌрЃЦрЃЋрЃљрЃЌ, рЃњрЃўрЃюрЃЊрЃљ input-рЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЊрЃљрЃфрЃЋрЃљ:

```python
def get_column():
    while True:
        try:
            c = input("WHICH COLUMN IS YOUR NUMBER IN? (1/2/3): ").strip()
            if c not in ["1", "2", "3"]:
                raise ValueError("Only 1, 2 or 3 allowed.")
            return int(c)
        except ValueError as e:
            print("ERROR:", e)
```

# 12. рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў (рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃарЃЌрЃБрЃџрЃў debugging)

рЃћрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў program compile/runРђЊрЃерЃў рЃљрЃа рЃЕрЃљрЃюрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```python
sum = 0
```

рЃћрЃА рЃљрЃарЃўрЃА **рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ** Рєњ рЃЎрЃЮрЃЊрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ.

Debugging-рЃўрЃА рЃњрЃќрЃљ:

```python
for i in range(1, 5):
    print("DEBUG i:", i)
    sum += i
```

   рЃЌрЃБ рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рђћ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ: РђърЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљРђю.

   * рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃАрЃўрЃљрЃА numbers.json-рЃўрЃЊрЃљрЃю
