# ­ЪЊў **р▓њр▓љр▓Ўр▓Ћр▓ћр▓Ќр▓ўр▓џр▓ў 01 Рђћ Python-р▓ўр▓А р▓Ар▓љр▓цр▓Бр▓Фр▓Ћр▓џр▓ћр▓Љр▓ў**


# 1. р▓ер▓ћр▓Ар▓љр▓Ћр▓љр▓џр▓ў Рђћ р▓ар▓љр▓А р▓Ћр▓љр▓Ўр▓ћр▓Ќр▓ћр▓Љр▓Ќ р▓Њр▓љ р▓ар▓љр▓бр▓Юр▓Џ

Python рЃљрЃарЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ:  
**рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃЏрЃљ рЃерЃћрЃФрЃџрЃЮрЃА рЃарЃЌрЃБрЃџрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃдрЃгрЃћрЃарЃљ.**

рЃћрЃА рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃљрЃарЃўрЃА рЃерЃћрЃюрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў.  
рЃЊрЃдрЃћрЃА рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ:

- рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ  
- рЃарЃљ рЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў (рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў)  
- рЃарЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљрЃљ рЃарЃўрЃфрЃ«рЃЋрЃАрЃљ рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃА рЃерЃЮрЃарЃўрЃА  
- рЃарЃЮрЃњрЃЮрЃа РђюрЃЋрЃћрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃћрЃЉрЃўрЃЌРђЮ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА  
- рЃЊрЃљрЃЋрЃгрЃћрЃарЃЌ рЃърЃўрЃарЃЋрЃћрЃџ рЃърЃљрЃбрЃљрЃарЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА  
- рЃњрЃљрЃЋрЃўрЃ«рЃАрЃћрЃюрЃћрЃЉрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў рЃерЃћрЃюрЃА рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў  
  *(21-рЃЎрЃљрЃарЃбрЃўрЃљрЃюрЃў рЃЌрЃљрЃЏрЃљрЃерЃў Рђћ `cart21.py`)*

# 2. р▓ар▓љр▓А р▓юр▓ўр▓ер▓юр▓љр▓Ћр▓А Рђюр▓ър▓ар▓Юр▓њр▓ар▓љр▓Џр▓ўр▓ар▓ћр▓Љр▓љРђЮ?

рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃЮ рЃЏрЃћрЃњрЃЮрЃЉрЃарЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃАрЃћ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃбрЃДрЃЋрЃў.  
рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЌрЃБ рЃЮрЃЊрЃюрЃљрЃЋ рЃњрЃљрЃБрЃарЃЎрЃЋрЃћрЃЋрЃџрЃљрЃЊ рЃћрЃБрЃЉрЃюрЃћрЃЉрЃў рЃарЃљрЃЏрЃћрЃА Рђћ рЃЕрЃћрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ:  
РђърЃЋрЃћрЃа рЃњрЃљрЃЋрЃўрЃњрЃћ, рЃерЃћрЃю рЃЏрЃћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃЌ рЃЏрЃўрЃЌрЃ«рЃљрЃарЃўРђю.

рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃ«рЃЊрЃћрЃЉрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃћрЃюрЃљрЃќрЃћ, рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃљрЃЊ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃЌрЃљрЃю рЃАрЃљрЃАрЃљрЃБрЃЉрЃарЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃюрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ.

рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃА, рЃ«рЃЮрЃџрЃЮ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃЏрЃ«рЃарЃўрЃЋ рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

> **рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃќрЃБрЃАрЃбрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃфрЃћрЃЏрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.**

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ рЃЉрЃћрЃЋрЃарЃўрЃљ рЃЊрЃљ рЃўрЃАрЃўрЃюрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА рЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃўрЃЌ рЃцрЃЮрЃарЃЏрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃ«рЃљрЃбрЃљрЃЋрЃћрЃю.

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљрЃА.

Python рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃћрЃарЃЌрЃў рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃћрЃЉрЃўрЃф рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃЊрЃЋрЃўрЃџрЃў рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃўрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.

# 3. р▓Ўр▓Юр▓Џр▓ћр▓юр▓бр▓љр▓ар▓ў - р▓Бр▓Џр▓юр▓ўр▓ер▓Ћр▓юр▓ћр▓џр▓Юр▓Ћр▓љр▓юр▓ћр▓Ар▓ў р▓љр▓бр▓ар▓ўр▓Љр▓Бр▓бр▓ў р▓ър▓ар▓Юр▓њр▓ар▓љр▓Џр▓ўр▓А р▓гр▓ћр▓ар▓ўр▓Ар▓љр▓А

рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃерЃў **рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃљ** Рђћ рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃЎрЃЮрЃЊрЃў рЃгрЃџрЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃњрЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃЮрЃА.

### **рЃарЃљрЃбрЃЮрЃЏрЃљрЃљ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў**

1. **рЃ«рЃАрЃюрЃўрЃА, рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃў**
   рЃЎрЃЮрЃЊрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃарЃў рЃўрЃДрЃЮрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃБрЃњрЃћрЃЉрЃљрЃарЃў. рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃњрЃљрЃюрЃЏрЃљрЃарЃЌрЃљрЃЋрЃА, рЃЌрЃБ *РђърЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљРђю*.

2. **рЃўрЃфрЃљрЃЋрЃА рЃфрЃЮрЃЊрЃюрЃљрЃА рЃЊрЃарЃЮрЃерЃў**
   рЃЊрЃдрЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃњрЃљрЃ«рЃАрЃЮрЃЋрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃћрЃарЃЌ рЃЌрЃЋрЃћрЃерЃў?
   рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃњрЃўрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃА.

3. **рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃЎрЃЮрЃџрЃљрЃЉрЃЮрЃарЃљрЃфрЃўрЃљрЃА**
   рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃљрЃарЃўрЃА рЃњрЃБрЃюрЃЊрЃБрЃарЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃюрЃљ.

4. **рЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃАрЃгрЃЮрЃа рЃЕрЃЋрЃћрЃЋрЃљрЃА**
   рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃгрЃћрЃарЃўрЃА рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃљ рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў РђърЃгрЃћрЃАрЃарЃўрЃњрЃўрЃљрЃюрЃўРђю рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃљрЃА.

5. **рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃърЃЮрЃБрЃџрЃЮрЃЉ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА**
   рЃарЃЮрЃфрЃљ рЃўрЃфрЃўрЃљрЃю, рЃарЃљ рЃБрЃюрЃЊрЃљ рЃ«рЃЊрЃћрЃЉрЃЮрЃЊрЃћрЃА, рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃљрЃЊрЃњрЃўрЃџрЃў рЃЏрЃљрЃџрЃћ рЃўрЃФрЃћрЃЉрЃюрЃћрЃЉрЃљ.

---

### **рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃў рЃцрЃЮрЃарЃЏрЃљ Python-рЃерЃў**

```python
# рЃћрЃА рЃЉрЃџрЃЮрЃЎрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃљрЃАрЃљрЃЎрЃЮрЃЉрЃарЃўрЃЋ рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃљрЃА
age = 12
if age >= 10:
    print("рЃерЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃћрЃА рЃЌрЃљрЃЏрЃљрЃерЃў рЃўрЃЌрЃљрЃЏрЃљрЃерЃЮ")
```

### **рЃарЃљ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў**

* рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЎрЃЮрЃЊрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃарЃЌрЃБрЃџ рЃўрЃЊрЃћрЃљрЃА
* рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃарЃўрЃфрЃ«рЃЋрЃА рЃљрЃю рЃАрЃбрЃарЃўрЃюрЃњрЃА рЃљрЃЦрЃЋрЃА **рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў рЃљрЃќрЃарЃў**
* рЃарЃЮрЃфрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ рЃљрЃа рЃЕрЃљрЃюрЃА рЃћрЃарЃЌрЃў рЃерЃћрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
* рЃарЃЮрЃфрЃљ рЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌ рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃА РђърЃЏрЃўрЃДрЃЋрЃљрЃюрЃћ рЃљрЃЦрЃљрЃЏрЃЊрЃћРђю

### **рЃарЃљ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃгрЃћрЃарЃЮ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃљрЃЊ**

* **рЃарЃљрЃф рЃЎрЃЮрЃЊрЃўрЃЊрЃљрЃю рЃўрЃАрЃћрЃЊрЃљрЃф рЃљрЃерЃЎрЃљрЃарЃљрЃљ**

рЃфрЃБрЃЊрЃў рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

```python
x = 5  # рЃљрЃЦ x-рЃА рЃЋрЃљрЃюрЃўрЃГрЃћрЃЉрЃЌ 5-рЃА
```

### **рЃерЃћрЃЏрЃљрЃ»рЃљрЃЏрЃћрЃЉрЃћрЃџрЃў рЃљрЃќрЃарЃў**

рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃљрЃарЃўрЃА **рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃљ рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃќрЃћ** Рђћ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃЌрЃљрЃЋрЃќрЃћ, рЃАрЃ«рЃЋрЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ. рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў рЃќрЃЮрЃњрЃљрЃЋрЃА рЃЊрЃарЃЮрЃА, рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА рЃЊрЃљ рЃЎрЃЮрЃЊрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃЎрЃБрЃџрЃбрЃБрЃарЃБрЃџ, рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃБрЃџ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА.


# 4. р▓фр▓Ћр▓џр▓љр▓Њр▓ў Рђћ р▓ър▓ўр▓ар▓Ћр▓ћр▓џр▓ў р▓Њр▓љ р▓Др▓Ћр▓ћр▓џр▓љр▓ќр▓ћ р▓Џр▓юр▓ўр▓ер▓Ћр▓юр▓ћр▓џр▓Юр▓Ћр▓љр▓юр▓ў р▓ўр▓юр▓Ар▓бр▓ар▓Бр▓Џр▓ћр▓юр▓бр▓ў

рЃфрЃЋрЃџрЃљрЃЊрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА **рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃЌрЃџрЃБрЃџ рЃДрЃБрЃЌрЃА**, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ.

рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃўрЃЊрЃљрЃю:

- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ РђърЃДрЃБрЃџрЃљрЃЉрЃљРђю Рђћ рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃцрЃБрЃџрЃА.  
- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў РђърЃАрЃљрЃЌрЃљрЃЊрЃљрЃарЃўрЃњрЃЮ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўРђю Рђћ рЃерЃўрЃњрЃюрЃўрЃЌ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃљ  
- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў РђърЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўРђю Рђћ рЃерЃўрЃњрЃюрЃўрЃЌ рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃўрЃљ

рЃДрЃЋрЃћрЃџрЃљ рЃљрЃЏ рЃДрЃБрЃЌрЃА рЃљрЃЦрЃЋрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ:
- **рЃАрЃљрЃ«рЃћрЃџрЃў**  
- **рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў**

рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў **рЃЮрЃа рЃюрЃљрЃгрЃўрЃџрЃљрЃЊ** рЃњрЃљрЃюрЃўрЃ«рЃўрЃџрЃћрЃЉрЃљ:

1. **рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃБрЃарЃў рЃгрЃћрЃАрЃў** Рђћ рЃарЃљрЃАрЃљрЃф рЃћрЃюрЃљ рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃЊ
2. **рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃерЃћрЃЌрЃљрЃюрЃ«рЃЏрЃћрЃЉрЃљ (рЃерЃћрЃцрЃБрЃЌрЃБрЃџрЃў рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃљ)** Рђћ рЃарЃљрЃАрЃљрЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃћрЃЉрЃў рЃЌрЃљрЃЋрЃљрЃЊ рЃўрЃфрЃљрЃЋрЃћрЃю, рЃарЃЮрЃЏ рЃЎрЃЮрЃЊрЃў рЃўрЃДрЃЮрЃА рЃЏрЃЎрЃљрЃцрЃўрЃЮ, рЃљрЃЎрЃБрЃарЃљрЃбрЃБрЃџрЃў рЃЊрЃљ рЃћрЃарЃЌрЃњрЃЋрЃљрЃарЃЮрЃЋрЃљрЃюрЃў

## **1. р▓Ар▓ўр▓юр▓бр▓љр▓Цр▓Ар▓Бр▓ар▓ў р▓гр▓ћр▓Ар▓ў (р▓Ар▓љр▓Ћр▓љр▓џр▓Њр▓ћр▓Љр▓Бр▓џр▓Ю р▓Џр▓Юр▓Ќр▓«р▓Юр▓Ћр▓юр▓ћр▓Љр▓ў Python-р▓ер▓ў)**

Python-рЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў **рЃБрЃюрЃЊрЃљ** рЃљрЃЎрЃЏрЃљрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃЊрЃћрЃА:

### Рюћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃАрЃЮрЃћрЃЉрЃў, рЃфрЃўрЃцрЃарЃћрЃЉрЃў рЃЊрЃљ рЃЦрЃЋрЃћрЃЊрЃљрЃбрЃўрЃарЃћ (`_`)

рЃЏрЃљрЃњ:

```
name
age2
user_name
```

### Рюћ рЃљрЃа рЃўрЃгрЃДрЃћрЃЉрЃљ рЃфрЃўрЃцрЃарЃўрЃЌ

РЮї `2name`
Рюћ `name2`

### Рюћ рЃЊрЃўрЃЊрЃў рЃЊрЃљ рЃърЃљрЃбрЃљрЃарЃљ рЃљрЃАрЃЮрЃћрЃЉрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃЊрЃћрЃЉрЃљ

`age`, `Age`, `AGE` Рђћ рЃАрЃљрЃЏрЃўрЃЋрЃћ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃўрЃљ.

### Рюћ рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ Python-рЃўрЃА "рЃЦрЃўрЃЋрЃЮрЃарЃЊрЃў"

рЃЏрЃљрЃњ: `if`, `else`, `for`, `class`, `def` рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.

### Рюћ рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃў

РЮї `name!`
РЮї `my-name`
РЮї `price$`

## **2. р▓ър▓ар▓Юр▓њр▓ар▓љр▓Џр▓ўр▓Ар▓бр▓ћр▓Љр▓ўр▓А р▓Џр▓ўр▓ћр▓а р▓Џр▓ўр▓др▓ћр▓Љр▓Бр▓џр▓ў р▓бр▓ар▓љр▓Њр▓ўр▓фр▓ўр▓Бр▓џр▓ў р▓ер▓ћр▓Ќр▓љр▓юр▓«р▓Џр▓ћр▓Љр▓љ**

рЃћрЃА рЃљрЃа рЃљрЃарЃўрЃА рЃЎрЃљрЃюрЃЮрЃюрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃўрЃА **рЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃўрЃЌрЃў рЃЎрЃБрЃџрЃбрЃБрЃарЃљ**, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃДрЃЋрЃћрЃџрЃљ рЃўрЃфрЃљрЃЋрЃА.

## **2.1 snake_case Рђћ Python-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў**

рЃДрЃЋрЃћрЃџрЃљ рЃАрЃўрЃбрЃДрЃЋрЃљ рЃърЃљрЃбрЃљрЃарЃљ рЃљрЃАрЃЮрЃћрЃЉрЃўрЃЌ + рЃЦрЃЋрЃћрЃЊрЃљрЃбрЃўрЃарЃћ:

```
user_name
start_time
max_speed
student_age
```

рЃћрЃА рЃљрЃарЃўрЃА Python-рЃўрЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃбрЃўрЃџрЃў (PEP8).
рЃърЃћрЃЊрЃљрЃњрЃЮрЃњрЃўрЃЎрЃўрЃА рЃЏрЃ«рЃарЃўрЃЋрЃљрЃф рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃюрЃљрЃЌрЃћрЃџрЃў рЃцрЃЮрЃарЃЏрЃљрЃљ.

---

## **2.2 рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃБрЃюрЃЊрЃљ рЃљрЃдрЃгрЃћрЃарЃЊрЃћрЃА рЃЏрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА**

рЃфрЃБрЃЊрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

```
x = 12
a = "рЃљрЃюрЃљ"
```

рЃАрЃгрЃЮрЃарЃў:

```
age = 12
student_name = "рЃљрЃюрЃљ"
lesson_count = 7
```

рЃЎрЃљрЃарЃњрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃў **рЃљрЃа рЃўрЃбрЃЮрЃЋрЃћрЃЉрЃА "рЃњрЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃЌ" рЃгрЃћрЃарЃўрЃА рЃАрЃбрЃўрЃџрЃА**.

## **2.3 camelCase Рђћ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Python-рЃерЃў**

рЃЌрЃБрЃЏрЃфрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃўрЃфрЃЮрЃЊрЃћ.

```
userName
```

рЃћрЃА рЃАрЃбрЃўрЃџрЃў рЃБрЃцрЃарЃЮ JavaScript-рЃерЃўрЃљ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў. Python-рЃерЃў **рЃљрЃа рЃљрЃарЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃфрЃЮрЃЊрЃюрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ.

## **2.4 CONSTANT_CASE Рђћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА**

## **3. р▓ар▓љр▓бр▓Юр▓Џ р▓љр▓ар▓ўр▓А р▓ћр▓А р▓љр▓Ар▓ћр▓Ќр▓ў р▓Џр▓юр▓ўр▓ер▓Ћр▓юр▓ћр▓џр▓Юр▓Ћр▓љр▓юр▓ў**

# **рЃЏрЃЮрЃЎрЃџрЃћ рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ (рЃЎрЃљрЃфрЃБрЃарЃљрЃЊ, рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа):**

**Python Рєњ snake_case.
рЃАрЃљрЃ«рЃћрЃџрЃў Рђћ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў.
рЃљрЃа рЃўрЃгрЃДрЃћрЃЉрЃљ рЃфрЃўрЃцрЃарЃўрЃЌ.
рЃљрЃа рЃўрЃДрЃћрЃюрЃћрЃЉ рЃЦрЃўрЃЋрЃЮрЃарЃЊрЃА.
рЃљрЃа рЃгрЃћрЃа рЃЉрЃљрЃюрЃФрЃў рЃЏрЃЮрЃЎрЃџрЃћ рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ.
Boolean рЃўрЃгрЃДрЃћрЃЉрЃљ is/has.
рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў Рєњ рЃЊрЃўрЃЊрЃў рЃљрЃАрЃЮрЃћрЃЉрЃўрЃЌ.**


Python-рЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃљрЃарЃўрЃА **рЃЉрЃЏрЃБрЃџрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃќрЃћ**, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЕрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃ«рЃћрЃарЃ«рЃћрЃЉрЃБрЃџ рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ рЃ░рЃЦрЃЋрЃўрЃљ.

# 4. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў Рђћ рЃарЃљрЃбрЃЮрЃЏрЃљрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?

Python-рЃА рЃБрЃюрЃЊрЃљ рЃўрЃфрЃЮрЃЊрЃћрЃА **рЃарЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃ░рЃЦрЃЮрЃюрЃЊрЃљ рЃљрЃЊрЃњрЃўрЃџрЃў**, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃўрЃбрЃЮрЃА рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃўрЃЦрЃфрЃћрЃА.

# 5. input() рЃЊрЃљ print() Рђћ рЃарЃЮрЃњрЃЮрЃа рЃЋрЃћрЃАрЃљрЃБрЃЉрЃарЃЮрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљ рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏ рЃЮрЃарЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА:

### print() Рђћ рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљ

```python
print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
```

### input() Рђћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃњрЃљрЃю рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ

```python
name = input("рЃарЃљ рЃњрЃЦрЃЋрЃўрЃљ? ")
```

рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:

1. рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА рЃбрЃћрЃЦрЃАрЃбрЃА
2. рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃерЃћрЃбрЃљрЃюрЃљрЃА
3. рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА `name` рЃфрЃЋрЃџрЃљрЃЊрЃерЃў

Рџа input() рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА TEXT-рЃА.
рЃљрЃюрЃБ `"25"` Рђћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃљ.

рЃЌрЃБ рЃарЃўрЃфрЃ«рЃЋрЃў рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ, рЃЏрЃљрЃерЃўрЃю рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў рЃфрЃ«рЃљрЃЊрЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃарЃЊрЃљрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃЏрЃЌрЃћрЃџ рЃарЃўрЃфрЃ«рЃЋрЃљрЃЊ рЃЊрЃљ рЃљрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `int()`.:

```python
age = int(input("рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃгрЃџрЃўрЃА рЃ«рЃљрЃа? "))
```
рЃћрЃА рЃЎрЃЮрЃЊрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃўрЃгрЃћрЃарЃЮрЃА рЃљрЃАрЃћрЃф:

```python
age = input("рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃгрЃџрЃўрЃА рЃ«рЃљрЃа? ")
age = int(age)
```
рЃљрЃЦ `age` рЃфрЃЋрЃџрЃљрЃЊрЃерЃў рЃ»рЃћрЃа рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃерЃћрЃЏрЃЮрЃбрЃљрЃюрЃўрЃџ рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃА, рЃ«рЃЮрЃџрЃЮ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃљрЃА рЃЋрЃљрЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃћрЃЉрЃЌ рЃЏрЃЌрЃћрЃџ рЃарЃўрЃфрЃ«рЃЋрЃерЃў.

рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃЎрЃЮрЃЊрЃўрЃА рЃЮрЃарЃўрЃЋрЃћ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃерЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА, рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА рЃарЃўрЃфрЃ«рЃЋрЃА. рЃљрЃЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃЋрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ рЃарЃЮрЃфрЃљ рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃА.

# 6. рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Рђћ рЃЊрЃљрЃгрЃћрЃарЃћ рЃЊрЃљ рЃњрЃљрЃўрЃњрЃћ

рЃЊрЃљрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў РђърЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃўРђю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ:

```python
name = input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃерЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃў: ")
age = int(input("рЃЊрЃљ рЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃарЃљрЃЏрЃЊрЃћрЃю рЃгрЃћрЃџрЃА? "))

print("рЃАрЃљрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃЮрЃљ рЃерЃћрЃюрЃў рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ,", name)
print("рЃерЃћрЃюрЃў рЃљрЃАрЃљрЃЎрЃўрЃљ:", age)
```

### рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃћрЃА?

# 8. рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃћрЃЉрЃерЃў

### РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ рЃфрЃўрЃцрЃарЃўрЃЌ

```python
1name = "Gio"   # рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ
```

### РЮї рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃерЃў

```python
city = Tbilisi   # рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃљ
```

### РЮї input()РђЊрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃљрЃЊ рЃљрЃарЃњрЃљрЃЊрЃљрЃЦрЃфрЃћрЃЋрЃљ

```python
age = input("age: ")  # рЃљрЃЦ age рЃљрЃарЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃарЃљ
```

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃгрЃЮрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљрЃљ:

```python
age = int(input("age: "))
```

# 9. р▓Ар▓«р▓Ћр▓љр▓Њр▓љр▓Ар▓«р▓Ћр▓љ р▓Ар▓ўр▓Фр▓юр▓ћр▓џр▓ўр▓А р▓ър▓љр▓бр▓љр▓ар▓љ р▓Ар▓љр▓Ћр▓љр▓ар▓»р▓ўр▓ер▓Юр▓ћр▓Љр▓ў 

## Level A Рђћ р▓Џр▓љр▓ар▓бр▓ўр▓Ћр▓ў

1. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃАрЃљрЃ«рЃћрЃџрЃА рЃЊрЃљ рЃБрЃгрЃћрЃарЃА: РђърЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ <рЃАрЃљрЃ«рЃћрЃџрЃў>Рђю
2. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃЦрЃљрЃџрЃљрЃЦрЃА рЃЊрЃљ рЃгрЃћрЃарЃА: РђърЃерЃћрЃю рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉ рЃЦрЃљрЃџрЃљрЃЦрЃерЃў <рЃЦрЃљрЃџрЃљрЃЦрЃў>Рђю.

## Level B Рђћ р▓Ар▓љр▓ер▓Бр▓љр▓џр▓Ю

3. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃ░рЃЎрЃўрЃЌрЃ«рЃћ: рЃАрЃљрЃ«рЃћрЃџрЃў, рЃњрЃЋрЃљрЃарЃў, рЃљрЃАрЃљрЃЎрЃў Рђћ рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃћрЃарЃЌ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃерЃў.
4. рЃЏрЃўрЃўрЃдрЃћ рЃЮрЃарЃў рЃарЃўрЃфрЃ«рЃЋрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃЏрЃљрЃЌрЃў рЃ»рЃљрЃЏрЃў.

## Level C Рђћ р▓ар▓Ќр▓Бр▓џр▓ў

5. рЃ░рЃЎрЃўрЃЌрЃ«рЃћ рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃгрЃћрЃџрЃў, рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃћ int-рЃљрЃЊ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃљрЃАрЃљрЃЎрЃў (2025РђЊрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃљрЃЎрЃћрЃџрЃў).
6. рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃдрЃћрЃЉрЃА 3 input-рЃА (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃљрЃАрЃљрЃЎрЃў, рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљ) рЃЊрЃљ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃЏрЃЮрЃЎрЃџрЃћ РђърЃърЃўрЃарЃљрЃЊ рЃЉрЃљрЃарЃљрЃЌрЃАРђю.

# 10. р▓ер▓ћр▓»р▓љр▓Џр▓ћр▓Љр▓љ

рЃљрЃЏ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃерЃў:

* рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў
* рЃњрЃљрЃЋрЃўрЃњрЃћрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў
* рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ input()/print()
* рЃЊрЃљрЃЋрЃгрЃћрЃарЃћрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ
* рЃерЃћрЃЋрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃћрЃЉрЃў 3 рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
