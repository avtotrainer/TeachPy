# 1. рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Рђћ рЃарЃљрЃА рЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ рЃЊрЃљ рЃарЃљрЃбрЃЮрЃЏ

Python рЃљрЃарЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ:  
**рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃЏрЃљ рЃерЃћрЃФрЃџрЃЮрЃА рЃарЃЌрЃБрЃџрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃдрЃгрЃћрЃарЃљ.**

рЃћрЃА рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃљрЃарЃўрЃА рЃерЃћрЃюрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў.  
рЃЊрЃдрЃћрЃА рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ:

- рЃарЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ  
- рЃарЃљ рЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў (рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў)  
- рЃарЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљрЃљ рЃарЃўрЃфрЃ«рЃЋрЃАрЃљ рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃА рЃерЃЮрЃарЃўрЃА  
- рЃарЃЮрЃњрЃЮрЃа РђюрЃЋрЃћрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃћрЃЉрЃўрЃЌРђЮ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА  
- рЃЊрЃљрЃЋрЃгрЃћрЃарЃЌ рЃърЃўрЃарЃЋрЃћрЃџ рЃърЃљрЃбрЃљрЃарЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА  
- рЃњрЃљрЃЋрЃўрЃ«рЃАрЃћрЃюрЃћрЃЉрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў рЃерЃћрЃюрЃА рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў  
  *(21-рЃЎрЃљрЃарЃбрЃўрЃљрЃюрЃў рЃЌрЃљрЃЏрЃљрЃерЃў Рђћ `cart21.py`)*

# 2. рЃарЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА РђюрЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљРђЮ?

рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃЮ рЃЏрЃћрЃњрЃЮрЃЉрЃарЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃАрЃћ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃбрЃДрЃЋрЃў.  
рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЌрЃБ рЃЮрЃЊрЃюрЃљрЃЋ рЃњрЃљрЃБрЃарЃЎрЃЋрЃћрЃЋрЃџрЃљрЃЊ рЃћрЃБрЃЉрЃюрЃћрЃЉрЃў рЃарЃљрЃЏрЃћрЃА Рђћ рЃЕрЃћрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ:  
РђърЃЋрЃћрЃа рЃњрЃљрЃЋрЃўрЃњрЃћ, рЃерЃћрЃю рЃЏрЃћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃЌ рЃЏрЃўрЃЌрЃ«рЃљрЃарЃўРђю.

рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃ«рЃЊрЃћрЃЉрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃћрЃюрЃљрЃќрЃћ, рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃљрЃЊ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃЌрЃљрЃю рЃАрЃљрЃАрЃљрЃБрЃЉрЃарЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃюрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ.

рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃАрЃљрЃБрЃЉрЃљрЃарЃў рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃА, рЃ«рЃЮрЃџрЃЮ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃЏрЃ«рЃарЃўрЃЋ рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

> **рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃќрЃБрЃАрЃбрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃфрЃћрЃЏрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.**

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ рЃЉрЃћрЃЋрЃарЃўрЃљ рЃЊрЃљ рЃўрЃАрЃўрЃюрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃўрЃЌ рЃцрЃЮрЃарЃЏрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃ«рЃљрЃбрЃљрЃЋрЃћрЃю.

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљрЃА.

Python рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃћрЃарЃЌрЃў рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃћрЃЉрЃўрЃф рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃЊрЃЋрЃўрЃџрЃў рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃўрЃљ.

# 3. рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў - рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃгрЃћрЃарЃўрЃАрЃљрЃА


# 4. рЃфрЃЋрЃџрЃљрЃЊрЃў Рђћ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў

рЃфрЃЋрЃџрЃљрЃЊрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА **рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃЌрЃџрЃБрЃџ рЃДрЃБрЃЌрЃА**, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ.

рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃўрЃЊрЃљрЃю:

- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ РђърЃДрЃБрЃџрЃљрЃЉрЃљРђю Рђћ рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃцрЃБрЃџрЃА.  
- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў РђърЃАрЃљрЃЌрЃљрЃЊрЃљрЃарЃўрЃњрЃЮ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўРђю Рђћ рЃерЃўрЃњрЃюрЃўрЃЌ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃљ  
- рЃњрЃљрЃЦрЃЋрЃА рЃДрЃБрЃЌрЃў РђърЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўРђю Рђћ рЃерЃўрЃњрЃюрЃўрЃЌ рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃўрЃљ

рЃДрЃЋрЃћрЃџрЃљ рЃљрЃЏ рЃДрЃБрЃЌрЃА рЃљрЃЦрЃЋрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ:
- **рЃАрЃљрЃ«рЃћрЃџрЃў**  
- **рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў**

Python-рЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃўрЃф рЃљрЃАрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА.

* `age` Рђћ рЃДрЃБрЃЌрЃў, рЃАрЃљрЃЊрЃљрЃф 25 рЃљрЃарЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў
* `name` Рђћ рЃДрЃБрЃЌрЃў, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃгрЃћрЃарЃўрЃљ РђъNikaРђю

Python-рЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃљрЃарЃўрЃА **рЃЉрЃЏрЃБрЃџрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃќрЃћ**, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЕрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃ«рЃћрЃарЃ«рЃћрЃЉрЃБрЃџ рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ рЃ░рЃЦрЃЋрЃўрЃљ.

рЃќрЃћрЃЏрЃЮрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЌрЃљрЃЋрЃўрЃАрЃЌрЃљрЃЋрЃљрЃЊрЃљрЃф рЃўрЃФрЃџрЃћрЃЋрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА, рЃЌрЃБ рЃарЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃА рЃЋрЃљрЃърЃўрЃарЃћрЃЉрЃЌ рЃЏрЃљрЃАрЃерЃў. рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃќрЃЮрЃњрЃљрЃЊрЃљрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃўрЃюрЃцрЃЮрЃарЃљрЃбрЃўрЃБрЃџрЃў, рЃарЃљрЃф рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў рЃЎрЃљрЃарЃњ рЃбрЃЮрЃюрЃљрЃЊ.

рЃќрЃћрЃЏрЃЮрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃЊрЃљрЃю `age` рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃЏрЃљрЃњрЃўрЃЋрЃарЃљрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў рЃљрЃа рЃњрЃЋрЃўрЃќрЃдрЃБрЃЊрЃљрЃЋрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ `a`, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЏрЃўрЃЌ рЃўрЃюрЃцрЃЮрЃарЃљрЃбрЃўрЃБрЃџрЃЮрЃЉрЃљ рЃерЃћрЃњрЃЋрЃћрЃќрЃдрЃБрЃЊрЃћрЃЉрЃљ, рЃ«рЃЮрЃџрЃЮ `name`-рЃА рЃЏрЃљрЃњрЃўрЃЋрЃарЃљрЃЊ, рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃБрЃцрЃарЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃБрЃџрЃў рЃўрЃДрЃЮрЃА `first_name`, рЃЌрЃБ рЃЏрЃљрЃАрЃерЃў рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃЋрЃљрЃА рЃЋрЃњрЃћрЃњрЃЏрЃљрЃЋрЃЌ.

рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃЊрЃљрЃю `is_student` рЃЊрЃљ `first_name` рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ, рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃљрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃЮрЃа рЃАрЃўрЃбрЃДрЃЋрЃљрЃА рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЊрЃљ рЃЏрЃљрЃЌ рЃЋрЃљрЃЉрЃљрЃЏрЃЌ рЃЦрЃЋрЃћрЃЊрЃљ рЃбрЃўрЃарЃћрЃЌрЃў, рЃћрЃА рЃБрЃЎрЃЋрЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃўрЃА рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃерЃў рЃърЃарЃЮрЃЉрЃћрЃџрЃўрЃА (рЃ░рЃљрЃарЃўрЃА) рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

рЃљрЃЦрЃЋрЃћ рЃЊрЃљрЃЋрЃљрЃќрЃБрЃАрЃбрЃЮрЃЌ рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃюрЃћрЃЉрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА:


рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃў!

Рџа `"25"` рЃљрЃарЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў
Рџа `25` рЃљрЃарЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў
**рЃћрЃА рЃЮрЃарЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА рЃљрЃа рЃБрЃЊрЃарЃўрЃА!**

## 4.4 рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў Рђћ `bool`

```python
logged_in = True
adult = False
```

рЃћрЃА рЃљрЃарЃўрЃА Python-рЃўрЃА РђърЃЊрЃўрЃљрЃ«Рђю рЃЊрЃљ РђърЃљрЃарЃљРђю.

# 5. input() рЃЊрЃљ print() Рђћ рЃарЃЮрЃњрЃЮрЃа рЃЋрЃћрЃАрЃљрЃБрЃЉрЃарЃЮрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљ рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏ рЃЮрЃарЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА:

### print() Рђћ рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљ

```python
print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
```

### input() Рђћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃњрЃљрЃю рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ

```python
name = input("рЃарЃљ рЃњрЃЦрЃЋрЃўрЃљ? ")
```

рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:

1. рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА рЃбрЃћрЃЦрЃАрЃбрЃА
2. рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃерЃћрЃбрЃљрЃюрЃљрЃА
3. рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА `name` рЃфрЃЋрЃџрЃљрЃЊрЃерЃў

Рџа input() рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА TEXT-рЃА.
рЃљрЃюрЃБ `"25"` Рђћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃљ.

рЃЌрЃБ рЃарЃўрЃфрЃ«рЃЋрЃў рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ:

```python
age = int(input("рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃгрЃџрЃўрЃА рЃ«рЃљрЃа? "))
```

# 6. рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Рђћ рЃЊрЃљрЃгрЃћрЃарЃћ рЃЊрЃљ рЃњрЃљрЃўрЃњрЃћ

рЃЊрЃљрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў РђърЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃўРђю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ:

```python
name = input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃерЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃў: ")
age = int(input("рЃЊрЃљ рЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃарЃљрЃЏрЃЊрЃћрЃю рЃгрЃћрЃџрЃА? "))

print("рЃАрЃљрЃАрЃўрЃљрЃЏрЃЮрЃЋрЃюрЃЮрЃљ рЃерЃћрЃюрЃў рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ,", name)
print("рЃерЃћрЃюрЃў рЃљрЃАрЃљрЃЎрЃўрЃљ:", age)
```

### рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃћрЃА?

1. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃњрЃЌрЃ«рЃЮрЃЋрЃА рЃАрЃљрЃ«рЃћрЃџрЃА
2. рЃњрЃљрЃЦрЃЋрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў `name`, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃбрЃћрЃЦрЃАрЃбрЃА
3. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃљрЃАрЃљрЃЎрЃА
4. `int()` рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃА рЃарЃўрЃфрЃ«рЃЋрЃљрЃЊ
5. рЃЉрЃЮрЃџрЃЮрЃА рЃЮрЃарЃўрЃЋрЃћрЃА рЃЉрЃљрЃќрЃљрЃарЃќрЃћ рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃА print()

рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА РђърЃАрЃбрЃљрЃарЃбрЃўрЃА рЃАрЃбрЃљрЃарЃбрЃўРђю.

# 7. рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃћрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў рЃерЃћрЃюрЃА 21-рЃЎрЃљрЃарЃбрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў?

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃў (рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃерЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў):

```
/mnt/data/cart21.py
```

рЃюрЃљрЃ«рЃћ рЃћрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў:

```python
c = input("WHICH COLUMN IS YOUR NUMBER IN? (1/2/3): ").strip()
```

рЃћрЃА рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃА input()РђЊрЃўрЃљ, рЃарЃљрЃАрЃљрЃф рЃљрЃ«рЃџрЃљ рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉ.
рЃЏрЃЮрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ рЃЌрЃљрЃЏрЃљрЃерЃерЃў рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃџрЃљрЃњрЃћрЃЉрЃџрЃљрЃЊ.

рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃўрЃЌ:

* **input** рЃўрЃдрЃћрЃЉрЃА рЃЎрЃЮрЃџрЃЮрЃюрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃА
* **рЃфрЃЋрЃџрЃљрЃЊрЃў c** рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃљрЃЏ рЃърЃљрЃАрЃБрЃ«рЃА
* **рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЏ рЃърЃљрЃАрЃБрЃ«рЃА рЃљрЃЊрЃљрЃарЃћрЃЉрЃА** рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ рЃ«рЃЊрЃћрЃЉрЃљ

рЃарЃљрЃф рЃЊрЃдрЃћрЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃћ Рђћ рЃерЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃљ.
рЃДрЃЋрЃћрЃџрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃљ input-рЃўрЃЌ рЃўрЃгрЃДрЃћрЃЉрЃљ.

# 8. рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃћрЃЉрЃерЃў

### РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ рЃфрЃўрЃцрЃарЃўрЃЌ

```python
1name = "Gio"   # рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ
```

### РЮї рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃерЃў

```python
city = Tbilisi   # рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃљ
```

### РЮї input()РђЊрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃљрЃЊ рЃљрЃарЃњрЃљрЃЊрЃљрЃЦрЃфрЃћрЃЋрЃљ

```python
age = input("age: ")  # рЃљрЃЦ age рЃљрЃарЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃарЃљ
```

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃгрЃЮрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљрЃљ:

```python
age = int(input("age: "))
```

# 9. рЃърЃљрЃбрЃљрЃарЃљ рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃћрЃЉрЃў (3 рЃЊрЃЮрЃюрЃћ)

## Level A Рђћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў

1. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃАрЃљрЃ«рЃћрЃџрЃА рЃЊрЃљ рЃБрЃгрЃћрЃарЃА: РђърЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ <рЃАрЃљрЃ«рЃћрЃџрЃў>Рђю
2. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃћрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃЦрЃљрЃџрЃљрЃЦрЃА рЃЊрЃљ рЃгрЃћрЃарЃА: РђърЃерЃћрЃю рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉ рЃЦрЃљрЃџрЃљрЃЦрЃерЃў <рЃЦрЃљрЃџрЃљрЃЦрЃў>Рђю.

## Level B Рђћ рЃАрЃљрЃерЃБрЃљрЃџрЃЮ

3. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃ░рЃЎрЃўрЃЌрЃ«рЃћ: рЃАрЃљрЃ«рЃћрЃџрЃў, рЃњрЃЋрЃљрЃарЃў, рЃљрЃАрЃљрЃЎрЃў Рђћ рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃћрЃарЃЌ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃерЃў.
4. рЃЏрЃўрЃўрЃдрЃћ рЃЮрЃарЃў рЃарЃўрЃфрЃ«рЃЋрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃЏрЃљрЃЌрЃў рЃ»рЃљрЃЏрЃў.

## Level C Рђћ рЃарЃЌрЃБрЃџрЃў

5. рЃ░рЃЎрЃўрЃЌрЃ«рЃћ рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃгрЃћрЃџрЃў, рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃћ int-рЃљрЃЊ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃљрЃАрЃљрЃЎрЃў (2025РђЊрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃљрЃЎрЃћрЃџрЃў).
6. рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃдрЃћрЃЉрЃА 3 input-рЃА (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃљрЃАрЃљрЃЎрЃў, рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљ) рЃЊрЃљ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃЏрЃЮрЃЎрЃџрЃћ РђърЃърЃўрЃарЃљрЃЊ рЃЉрЃљрЃарЃљрЃЌрЃАРђю.

# 10. рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

рЃљрЃЏ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃЌ:

* рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў
