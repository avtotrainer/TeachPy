
# Lesson 02 — ᲢᲔᲥᲡᲢᲘᲡ ᲛᲘᲦᲔᲑᲐ (input)

## ᲠᲐᲡ ᲕᲡᲬᲐᲕᲚᲝᲑᲗ
- როგორ ვავალებთ კომპიუტერს ტექსტის მიღებას
- როგორ ინახავს კომპიუტერი მიღებულ ინფორმაციას ცვლადში
- როგორ გამოვიტანოთ მიღებული პასუხი ეკრანზე

---

## ᲠᲐ ᲐᲠᲘᲡ input()
input() არის ბრძანება, რომელიც კომპიუტერს ეუბნება:

**„აიღე ის ტექსტი, რასაც ბავშვი აკრეფს.“**


```python
name = input("ENTER NAME: ")
```

ეს მაგალითი ეკრანზე გამოინტანს "ENTER NAME: " და დაელოდება შეიყვანო კლავიატურიდან რამე, რასაც ის შეინახავს ცვლადში `name`.

## დავალება

შეიყვანე ზუსტად ეს ტექსტი:

```python
name = input("ENTER NAME: ")
print("HELLO ",  name)
```
