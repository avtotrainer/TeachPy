## р▓ар▓љ р▓љр▓ар▓ўр▓А input()
input() рЃљрЃарЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ:

**РђърЃљрЃўрЃдрЃћ рЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃљрЃАрЃљрЃф рЃЉрЃљрЃЋрЃерЃЋрЃў рЃљрЃЎрЃарЃћрЃцрЃА.Рђю**


```python
name = input("ENTER NAME: ")
```

рЃћрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃюрЃбрЃљрЃюрЃА "ENTER NAME: " рЃЊрЃљ рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃЮ рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃЊрЃљрЃю рЃарЃљрЃЏрЃћ, рЃарЃљрЃАрЃљрЃф рЃўрЃА рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃфрЃЋрЃџрЃљрЃЊрЃерЃў `name`.


````md
# рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 02 Рђћ рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃърЃўрЃарЃЮрЃЉрЃўрЃЌрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў (if)

рЃљрЃЏ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃерЃў рЃерЃћрЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ:
- Python-рЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃбрЃўрЃърЃА  
- рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃћрЃЉрЃў  
- if/elif/else рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА  
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃдрЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃљрЃА  
- рЃарЃЮрЃњрЃЮрЃа рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ рЃерЃћрЃюрЃА 21-рЃЎрЃљрЃарЃбрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў (cart21.py)

# 1. рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Рђћ рЃарЃљрЃбрЃЮрЃЏ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў?

рЃЌрЃБ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃњрЃЋрЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃЊрЃљ, рЃарЃЮрЃњрЃЮрЃа *рЃЋрЃћрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃћрЃЉрЃўрЃЌ* рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА,  
рЃЏрЃљрЃерЃўрЃю рЃЏрЃћрЃЮрЃарЃћ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃњрЃЋрЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА, **рЃарЃЮрЃњрЃЮрЃа рЃЋрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃюрЃћрЃЉрЃЌ** рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА.

рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ:
> РђюрЃЏрЃўрЃЌрЃ«рЃљрЃарЃў рЃарЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮ рЃљрЃЏ рЃЮрЃа рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА рЃерЃЮрЃарЃўрЃА? рЃерЃћрЃљрЃЊрЃљрЃарЃћ? рЃерЃћрЃЎрЃарЃўрЃЉрЃЮ? рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћ?РђЮ

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
- рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЏрЃљ рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃљ рЃљрЃАрЃљрЃЎрЃў  
- рЃЕрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌ: рЃљрЃарЃўрЃА рЃўрЃА >= 18?  
- рЃљрЃю рЃЌрЃљрЃЏрЃљрЃерЃерЃў: рЃЕрЃљрЃцрЃўрЃЦрЃарЃћрЃЉрЃБрЃџрЃў рЃЉрЃљрЃарЃљрЃЌрЃў рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџ рЃАрЃЋрЃћрЃбрЃерЃўрЃљ?

рЃћрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃАрЃгрЃЮрЃарЃћрЃЊ рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА.

# 2. рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў Рђћ рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў

Python-рЃерЃў рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃљрЃџрЃЎрЃБрЃџрЃљрЃбрЃЮрЃарЃерЃў:

```python
a = 10 + 5      # 15
b = 10 - 3      # 7
c = 4 * 2       # 8
d = 8 / 2       # 4.0 (рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА float)
e = 10 // 3     # 3  (рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ)
f = 10 % 3      # 1  (рЃюрЃљрЃерЃЌрЃў)
g = 2 ** 3      # 8  (рЃ«рЃљрЃарЃўрЃАрЃ«рЃў)
````

рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃў рЃљрЃЏ рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃА рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА.

# 3. рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў Рђћ True/False рЃерЃћрЃЊрЃћрЃњрЃў

рЃћрЃА рЃюрЃљрЃгрЃўрЃџрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ `if`РђЊрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

| рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў | рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ      |
| --------- | ---------------- |
| ==        | рЃбрЃЮрЃџрЃўрЃљ?           |
| !=        | рЃљрЃа рЃљрЃарЃўрЃА рЃбрЃЮрЃџрЃў?    |
| >         | рЃЏрЃћрЃбрЃўрЃљ            |
| <         | рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ         |
| >=        | рЃЏрЃћрЃбрЃўрЃљ рЃљрЃю рЃбрЃЮрЃџрЃў    |
| <=        | рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ рЃљрЃю рЃбрЃЮрЃџрЃў |

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```python
age = 20
print(age >= 18)  # True
print(age == 30)  # False
print(age != 10)  # True
```

рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ **рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА True рЃљрЃю False**.

# 4. рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў Рђћ and, or, not

рЃћрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃарЃЌрЃБрЃџрЃў рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

* `and` Рђћ рЃЮрЃарЃўрЃЋрЃћ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА True
* `or` Рђћ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃЏрЃљрЃўрЃюрЃф рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА True
* `not` Рђћ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃАрЃљрЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃЮрЃА

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```python
age = 25
has_id = True

print(age >= 18 and has_id)   # True
print(age >= 18 and not has_id) # False
print(age >= 18 or has_id)    # True
```

# 5. рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ Рђћ if, elif, else

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃћрЃюрЃбрЃўрЃЊрЃљрЃю, рЃарЃЮрЃфрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА РђюрЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃюрЃћрЃЉрЃљрЃАРђЮ рЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃўрЃюрЃћрЃЉрЃЌ.

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ:

```python
if condition:
    рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ
elif another_condition:
    рЃЏрЃћрЃЮрЃарЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ
else:
    рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ
```

# 12. рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

* if/elif/else

рЃћрЃА рЃфрЃЮрЃЊрЃюрЃљ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА:

­ЪЉЅ **рЃфрЃўрЃЎрЃџрЃћрЃЉрЃў (for, while) Рђћ Python-рЃўрЃА рЃњрЃБрЃџрЃў рЃЊрЃљ рЃАрЃБрЃџрЃў**
