# 4. return — შედეგის დაბრუნება

ფუნქცია ყოველთვის *აბრუნებს* რაღაცას.
თუ არ დავწერთ return-ს → აბრუნებს None-ს.

```python
def add(a, b):
    return a + b

x = add(3, 5)
print(x)
```

return:

* აბრუნებს მონაცემს,
* წყვეტს ფუნქციის მუშაობას.

# 8. Type hints — პროფესიული სტანდარტი

ფუნქციებს შეიძლება დავამატოთ type hints:

# 9. Docstrings — ფუნქციის ოფიციალური აღწერა

Python სტანდარტი:

```python
def multiply(a: int, b: int) -> int:
    """
    აძლიერებს ორ რიცხვს და აბრუნებს შედეგს.
    """
    return a * b
```

დეველოპერებს ეხმარება ადგილზევე გავიგოთ ფუნქციის დანიშნულება.
