````md
# рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў 04 Рђћ рЃАрЃўрЃљрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў Python-рЃерЃў: list, dict, tuple, set

# 1. рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Рђћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў

рЃгрЃўрЃюрЃљ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃерЃў рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃфрЃўрЃЎрЃџрЃћрЃЉрЃў.  
рЃљрЃ«рЃџрЃљ рЃЏрЃўрЃЋрЃЊрЃўрЃЋрЃљрЃарЃЌ рЃўрЃЦ, рЃарЃљрЃАрЃљрЃф Python-рЃА рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ *рЃФрЃџрЃўрЃћрЃа рЃћрЃюрЃЮрЃЉрЃљрЃЊ* рЃљрЃЦрЃфрЃћрЃЋрЃА Рђћ  
**рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃћрЃЉрЃў**.

рЃарЃЮрЃфрЃљ рЃњрЃўрЃюрЃЊрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮ:

- рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃў,
- рЃљрЃю рЃЌрЃБрЃюрЃЊрЃљрЃф рЃџрЃЮрЃњрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃћрЃЉрЃў,

| tuple | рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃарЃўрЃњрЃў | рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ, рЃАрЃгрЃарЃљрЃцрЃў |

# 2. List Рђћ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

## 2.1 рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃўрЃљ?

рЃАрЃўрЃљ рЃљрЃарЃўрЃА **рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃарЃўрЃњрЃў**, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃљрЃЦрЃЋрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃў.  
рЃўрЃюрЃЊрЃћрЃЦрЃАрЃљрЃфрЃўрЃљ рЃўрЃгрЃДрЃћрЃЉрЃљ **0-рЃўрЃЊрЃљрЃю**.

```python
cards = [1, 2, 3, 4, 5]
````

рЃћрЃА рЃљрЃарЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃљрЃю рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃћрЃарЃўрЃљ.

## 2.2 рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ

```python
print(cards[0])  # 1
print(cards[3])  # 4
```

## 2.3 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ

```python
cards[1] = 99
```

## 2.4 рЃАрЃўрЃўрЃА рЃгрЃарЃћрЃќрЃћ рЃњрЃљрЃЋрЃџрЃљ (iteration)

```python
for card in cards:
    print(card)
```

## 2.5 рЃАрЃўрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў

```python
a = [3, 1, 2]
a.append(10)      # [3,1,2,10]
a.insert(0, 99)   # [99,3,1,2,10]
a.remove(1)       # [99,3,2,10]
a.sort()          # [2,3,10,99]
a.reverse()       # [99,10,3,2]
```

# 3. List slicing Рђћ рЃФрЃџрЃўрЃћрЃарЃў рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў

Python-рЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў.

```python
numbers = [10,20,30,40,50]
print(numbers[1:4])   # [20,30,40]
```

рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃљ:

```
[start : end : step]
```

## рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:

```python
a = [0,1,2,3,4,5,6]

a[1:5]    # [1,2,3,4]
a[::2]    # [0,2,4,6]
a[::-1]   # рЃАрЃўрЃљрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃбрЃарЃўрЃљрЃџрЃћрЃЉрЃБрЃџрЃљрЃЊ
```

---

# 4. рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў slicing рЃерЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў (21-рЃЎрЃљрЃарЃбрЃўрЃљрЃюрЃў рЃЌрЃљрЃЏрЃљрЃерЃў)

рЃерЃћрЃюрЃА рЃцрЃљрЃўрЃџрЃерЃў (cart21.py) рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ slicing-рЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљ:
**рЃДрЃЮрЃЋрЃћрЃџрЃў рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ**:

```python
col1 = cards[0::3]
col2 = cards[1::3]
col3 = cards[2::3]
```

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

* col1 Рєњ рЃўрЃдрЃћрЃЉрЃА 0РђЊрЃЊрЃљрЃю рЃЊрЃљрЃгрЃДрЃћрЃЉрЃБрЃџ рЃДрЃЋрЃћрЃџрЃљ 3-рЃћрЃБрЃџрЃА
* col2 Рєњ рЃўрЃдрЃћрЃЉрЃА 1РђЊрЃЊрЃљрЃю рЃДрЃЮрЃЋрЃћрЃџрЃў рЃЏрЃћрЃАрЃљрЃЏрЃћ
* col3 Рєњ рЃўрЃдрЃћрЃЉрЃА 2РђЊрЃЊрЃљрЃю рЃДрЃЮрЃЋрЃћрЃџрЃў рЃЏрЃћрЃАрЃљрЃЏрЃћ

рЃћрЃА рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА **21 рЃЉрЃљрЃарЃљрЃЌрЃў 3 рЃАрЃЋрЃћрЃбрЃљрЃЊ рЃЊрЃљрЃЋрЃДрЃЮрЃЌ рЃќрЃБрЃАрЃбрЃљрЃЊ 7-7-7 рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.**

# 5. Dictionary Рђћ рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃў/рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ

рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў рЃљрЃарЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЋрЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃЌ **рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃЌ**, рЃљрЃарЃљ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃЌ.

```python
person = {
    "name": "Nika",
    "age": 21,
    "city": "Tbilisi"
}
```

## рЃгрЃЋрЃЊрЃЮрЃЏрЃљ:

```python
print(person["name"])    # Nika
```

## рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ:

```python
person["job"] = "engineer"
```

## get() Рђћ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ:

```python
print(person.get("salary"))  # None
```

## iteration:

```python
for key, value in person.items():
    print(key, value)
```

рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃўрЃЌ:

* dict рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ **hash table** рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃА Рєњ рЃФрЃћрЃЉрЃюрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃгрЃарЃљрЃцрЃўрЃљ.

# 6. Tuple Рђћ рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃАрЃўрЃљ

Tuple рЃљрЃарЃўрЃА list, рЃЏрЃљрЃњрЃарЃљрЃЏ **рЃерЃћрЃБрЃфрЃЋрЃџрЃћрЃџрЃў**.

```python
point = (10, 20)
```

рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?

* tuple рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃюрЃљрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЊрЃљ рЃљрЃдрЃљрЃа рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ
* рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ
* рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:

* рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў
* рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃћрЃЉрЃў
* рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў

# 7. Set Рђћ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ

Set рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃерЃЮрЃарЃћрЃЉрЃА рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃарЃўрЃА РђърЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃўРђю рЃюрЃљрЃЎрЃарЃћрЃЉрЃў.

```python
s = {1,2,3,3,4}
print(s)  # {1,2,3,4}
```

set-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃ«рЃЊрЃћрЃЉрЃљ:

* рЃгрЃћрЃЋрЃарЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ: `"apple" in s`
* duplicates removal
* union, intersection

# 9. рЃарЃљрЃбрЃЮрЃЏ рЃБрЃюрЃЊрЃљ рЃЋрЃўрЃфрЃЮрЃЊрЃћ рЃћрЃА 21-рЃЎрЃљрЃарЃбрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА?

рЃърЃарЃЮрЃћрЃЦрЃбрЃў heavily рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ list-рЃА:

* рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ list-рЃерЃў

* рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃў 3 рЃАрЃЋрЃћрЃбрЃљрЃЊ рЃўрЃДрЃЮрЃцрЃљ slicing-рЃўрЃЌ
* рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃА рЃАрЃЋрЃћрЃбрЃћрЃЉрЃў рЃћрЃарЃЌ list-рЃерЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ
* рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃА 11-рЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў (index 10) **рЃљрЃарЃўрЃА player's рЃАрЃљрЃДрЃЋрЃљрЃарЃћрЃџрЃў рЃЉрЃљрЃарЃљрЃЌрЃў**
