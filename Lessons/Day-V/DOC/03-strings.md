# ✔️ Day 5 → Concept 2

## **Slicing — სტრიქონის ნაწილის ამოჭრა**

Slicing მუშაობს შემდეგი ფორმით:

```
text[start : end]
```

* `start` — საიდან დავიწყოთ
* `end` — სადამდე მივიდეს (მხოლოდ *მუშაობამდე*)
* თუ რომელიმე გამოტოვებულია, Python თვითონ მიხვდება

მაგალითად:

```python
name = "Python"

name[0:3]   # "Pyt"
name[2:]    # "thon"
name[:4]    # "Pyth"
```

ეს ძალიან ბევრს გამოიყენება:
ფაილების სახელების ამოჭრა, თარიღების გამოტანა, ტექსტის ფორმატირება და ა.შ.
