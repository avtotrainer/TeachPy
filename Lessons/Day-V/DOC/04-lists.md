# Рюћ№ИЈ Day 5 Рєњ Concept 3

## **List Comprehension Рђћ рЃАрЃўрЃўрЃА рЃАрЃгрЃарЃљрЃцрЃў рЃерЃћрЃЦрЃЏрЃюрЃљ**

рЃћрЃА рЃљрЃарЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљ рЃФрЃџрЃўрЃћрЃарЃў рЃњрЃќрЃљ рЃАрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ.

### рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў рЃњрЃќрЃљ:

```python
numbers = [1, 2, 3, 4, 5]
squares = []
for n in numbers:
    squares.append(n * n)
```

### рЃўрЃњрЃўрЃЋрЃћ List Comprehension-рЃўрЃЌ:

```python
squares = [n * n for n in numbers]
```

### рЃцрЃўрЃџрЃбрЃарЃўрЃЌ:

```python
even = [n for n in numbers if n % 2 == 0]
```

рЃћрЃА рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

* рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљрЃерЃў
* рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљрЃерЃў
* рЃбрЃћрЃЦрЃАрЃбрЃћрЃЉрЃўрЃА рЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљрЃерЃў
* JSON рЃЊрЃљ CSV рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃерЃў
