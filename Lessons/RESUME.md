# Python Рђћ 7-Day Summary

###  7 рЃЊрЃдрЃўрЃљрЃюs Python-рЃўрЃА рЃЏрЃЮрЃЎрЃџрЃћ рЃЎрЃБрЃарЃАрЃў рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.

## ­ЪЊў р▓Њр▓др▓ћ I Рђћ Basics

#### рЃФрЃўрЃарЃћрЃБрЃџрЃў рЃфрЃюрЃћрЃЉрЃћрЃЉрЃў

- рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (variables)
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў (str, int, float, bool)
- рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў (+, -, *, /)
- Input / Output  
- рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ (`int()`, `str()`)

---

## ­ЪЊў Day 2 Рђћ Control Flow
- if / elif / else  
- for рЃфрЃўрЃЎрЃџрЃў (`for i in range()`)
- while рЃфрЃўрЃЎрЃџрЃў  
- try / except  
- рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ (`ValueError`)

---

## ­ЪЊў Day 3 Рђћ Data Structures
- List Рђћ рЃерЃћрЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃўрЃљ  
- Tuple Рђћ рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃАрЃўрЃљ  
- Dictionary Рђћ key/value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃў  
- Set Рђћ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў  
- iteration over dict (`.items()`)

---

## ­ЪЊў Day 4 Рђћ Functions, Modules, Files
- рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (`def`), рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў, `return`
- рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў (`import`, `from import`)
- рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ/рЃЕрЃљрЃгрЃћрЃарЃљ (`open`, `with open`)

---

## ­ЪЊў Day 5 Рђћ Advanced Data Manipulation
- рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў: `upper`, `lower`, `title`, `replace`
- slicing (`text[:]`)
- split / join
- List Comprehension
- datetime: `now()`, `strftime`, `strptime`, `timedelta`

---

## ­ЪЊў Day 6 Рђћ Object-Oriented Programming (OOP)
- Class Рђћ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃерЃљрЃЉрЃџрЃЮрЃюрЃў  
- Object Рђћ рЃЎрЃџрЃљрЃАрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў  
- `__init__` рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃў  
- Attributes (`self.name`)
- Methods  
- Practical example: Rectangle class

---

## ­ЪЊў Day 7 Рђћ JSON, Debugging, AI Basics
- JSON рЃЕрЃљрЃгрЃћрЃарЃљ (`json.dump`)  
- JSON рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ (`json.load`)

- NumPy Рђћ array рЃЊрЃљ рЃЋрЃћрЃЦрЃбрЃЮрЃарЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў

## Рюћ№ИЈ рЃерЃћрЃЊрЃћрЃњрЃў
