# Python – Day 1 (Basics)

## 1. Variables
Python-ში ცვლადი არის სახელწოდება, რომელსაც მიენიჭა მნიშვნელობა.

```python
name = "Nika"
age = 12
```

## 2. Data Types
- str (ტექსტი)
- int (მთელი რიცხვი)
- float (ათწილადი)
- bool (True / False)

## 3. Operators
- `+`  დამატება
- `-`  გამოკლება
- `*`  გამრავლება
- `/`  გაყოფა

## 4. Input
```python
name = input("სახელი: ")
```

## 5. Output
```python
print("გამარჯობა,", name)
```

## 6. Type Conversion
```python
age = int(input("ასაკი: "))
```
