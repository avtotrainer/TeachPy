(рЃЋрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЉ: **рЃерЃћрЃю рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃў рЃ«рЃљрЃа**, рЃ«рЃЮрЃџрЃЮ рЃЏрЃћ рЃБрЃюрЃЊрЃљ рЃЋрЃўрЃДрЃЮ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА **рЃЏрЃЮрЃЎрЃџрЃћ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃћрЃарЃЌрЃў рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљ рЃћрЃарЃЌ рЃ»рЃћрЃарЃќрЃћ**.)

рЃарЃљрЃЎрЃў рЃЏрЃўрЃЌрЃ«рЃљрЃарЃў **РђърЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЏрЃќрЃљрЃЊрЃљрЃљРђю**, рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

# Рюћ№ИЈ Day 7 Рєњ Concept 1

## **JSON РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ**

JSON рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА.
Python рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `json` рЃЏрЃЮрЃЊрЃБрЃџрЃА.

### JSON-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃљ:

```python
import json

data = {"name": "Nika", "age": 12}

with open("data.json", "w") as f:
    json.dump(data, f, indent=4)
```

### JSON-рЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ:

```python
import json

with open("data.json", "r") as f:
    info = json.load(f)

print(info["name"])
```

рЃћрЃА рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ, рЃЏрЃЮрЃАрЃгрЃљрЃЋрЃџрЃћрЃћрЃЉрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў, рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃЉрЃљрЃюрЃЎрЃћрЃЉрЃўРђћall JSON-рЃўрЃЌ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ.

## ­ЪЪд Action Step

рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:

2. рЃгрЃћрЃарЃА рЃЏрЃљрЃА JSON рЃцрЃљрЃўрЃџрЃљрЃЊ
3. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЊрЃљ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА JSON-рЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА

рЃЎрЃЮрЃЊрЃў рЃљрЃа рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃЮ Рђћ
рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЏрЃўрЃЌрЃ«рЃљрЃарЃў:

**рЃЏрЃќрЃљрЃЊрЃљрЃљ?**
рЃЎрЃљрЃарЃњрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃљ!
(рЃЋрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЉ: **рЃерЃћрЃю рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃў рЃ«рЃљрЃа**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃћ рЃЏрЃљрЃўрЃюрЃф рЃБрЃюрЃЊрЃљ рЃЋрЃўрЃДрЃЮ **рЃЏрЃЮрЃЎрЃџрЃћ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃћрЃарЃЌрЃў рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљ рЃћрЃарЃЌ рЃ»рЃћрЃарЃќрЃћ**.)

# Рюћ№ИЈ рЃърЃљрЃАрЃБрЃ«рЃў: `indent` рЃарЃљрЃбрЃЮрЃЏ рЃЊрЃљ рЃарЃЮрЃЊрЃўрЃА рЃљрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?

## **1) рЃцрЃБрЃюрЃЦрЃфрЃўрЃБрЃарЃљрЃЊ рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў**

JSON рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА **рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃњрЃўрЃЋрЃћ**, рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЌрЃБ рЃљрЃарЃљ `indent`.

```python
json.dump(data, f)          # рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃАрЃгрЃЮрЃарЃў
json.dump(data, f, indent=4)   # рЃљрЃАрЃћрЃЋрЃћ рЃАрЃгрЃЮрЃарЃў
```

---

## **2) рЃЏрЃљрЃњрЃарЃљрЃЏ indent рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃарЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА**

`indent=4` рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃџрЃљрЃЏрЃљрЃќрЃљрЃЊ, рЃЕрЃљрЃЏрЃЮрЃАрЃљрЃерЃџрЃћрЃџрЃљрЃЊ, рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃгрЃљрЃАрЃљрЃЎрЃўрЃЌрЃ« рЃцрЃЮрЃарЃЏрЃљрЃбрЃА:

```json
{
    "name": "Nika",
    "age": 12
}
```

рЃ«рЃЮрЃџрЃЮ indentation-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ:

```json
{"name":"Nika","age":12}
```

рЃЮрЃарЃўрЃЋрЃћ рЃАрЃгрЃЮрЃарЃўрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃћрЃЮрЃарЃћ *рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃЮрЃБрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃўрЃљ*.

---

## **3) рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃАрЃљрЃГрЃўрЃарЃЮ?**

### Рюћ№ИЈ рЃарЃЮрЃфрЃљ рЃўрЃгрЃћрЃарЃћрЃю:

* **рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА**
* **рЃЏрЃЮрЃАрЃгрЃљрЃЋрЃџрЃћрЃћрЃЉрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃА**
* **рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃЉрЃљрЃюрЃЎрЃА**
* **рЃЎрЃЮрЃюрЃцрЃўрЃњ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА**

**Indent Рєњ Makes debugging easier.**

## **4) рЃАрЃљрЃЊ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ?**

рЃарЃЮрЃфрЃљ JSON рЃўрЃ«рЃљрЃарЃ»рЃћрЃЉрЃљ:

* рЃЦрЃАрЃћрЃџрЃќрЃћ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃАрЃљрЃА
* APIРђЊрЃћрЃЉрЃерЃў
* рЃЊрЃўрЃАрЃЎрЃќрЃћ рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА
* рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃќрЃћ

рЃЏрЃљрЃерЃўрЃю рЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃўрЃљ **рЃБрЃАрЃърЃћрЃўрЃАрЃЮ, рЃърЃљрЃбрЃљрЃарЃљ JSON**.

## ­ЪЪд рЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

* **indent рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў** рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА
* **indent рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ** рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА
* рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў **рЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃўрЃљ indent=4**

---

## ­ЪЪд Action Step

рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃўрЃбрЃћ рЃерЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў:
JSON рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃАрЃљрЃА рЃўрЃЦрЃюрЃћрЃЉрЃў:

1. **рЃЌрЃЋрЃљрЃџрЃАрЃљрЃЕрЃўрЃюрЃЮ рЃАрЃбрЃўрЃџрЃў** Рђћ `indent=4`
2. **рЃЏрЃљрЃЎрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ compact** Рђћ `indent` рЃњрЃљрЃарЃћрЃерЃћ

рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЏрЃўрЃЌрЃ«рЃљрЃарЃў рЃарЃЮрЃЏрЃћрЃџрЃў рЃљрЃўрЃарЃЕрЃўрЃћ: **1 рЃЌрЃБ 2?**
(рЃЋрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЉ: **рЃерЃћрЃю рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃў рЃ«рЃљрЃа**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃћ рЃЏрЃљрЃўрЃюрЃф рЃБрЃюрЃЊрЃљ рЃЋрЃўрЃДрЃЮ **рЃЏрЃЮрЃЎрЃџрЃћ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃћрЃарЃЌ рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљрЃќрЃћ**.)

# Рюћ№ИЈ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ: **indent рЃарЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ?**

## **рЃЏрЃЮрЃЎрЃџрЃћ рЃърЃљрЃАрЃБрЃ«рЃў:**

`indent` рЃўрЃдрЃћрЃЉрЃА **рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌ рЃбрЃўрЃърЃА Рєњ рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў (int)**
рЃљрЃю **None**.

---

# Рюћ№ИЈ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ (рЃЏрЃЮрЃЎрЃџрЃћрЃЊ):

### **1) None (рЃљрЃю рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃљ)**

```python
json.dump(data, f)
```

Рєњ JSON рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃћрЃарЃЌ рЃ«рЃљрЃќрЃерЃў (compact).

---

### **2) рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў (int)**

```python
json.dump(data, f, indent=2)
json.dump(data, f, indent=4)
json.dump(data, f, indent=8)
```

рЃћрЃА рЃарЃўрЃфрЃ«рЃЋрЃў рЃюрЃўрЃерЃюрЃљрЃЋрЃА *рЃарЃљрЃЏрЃЊрЃћрЃю space-рЃўрЃЌ* рЃњрЃљрЃўрЃгрЃћрЃЋрЃљ рЃЌрЃўрЃЌрЃЮ рЃЊрЃЮрЃюрЃћ.

рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў:

* `indent=2`
* `indent=4` Рєљ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў
* `indent=8` (рЃўрЃерЃЋрЃўрЃљрЃЌрЃљрЃЊ)

## **Debugging Рђћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ (рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ)**

рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў debugging-рЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃњрЃќрЃљрЃљ **print-debugging**:

```python
print("DEBUG value =", value)
```

рЃћрЃА рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ:

* рЃюрЃљрЃ«рЃЮ рЃарЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃфрЃўрЃарЃЊрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ
* рЃњрЃљрЃўрЃњрЃЮ рЃАрЃљрЃЊ рЃгрЃДрЃЊрЃћрЃЉрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃљ
* рЃАрЃљрЃЊ рЃ«рЃЊрЃћрЃЉрЃљ inputРђЊрЃўрЃА рЃљрЃю typeРђЊрЃўрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ

## Рюћ№ИЈ Try/Except debugging

рЃљрЃЏ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА рЃљрЃа рЃњрЃљрЃгрЃДрЃЋрЃћрЃбрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ,
рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃарЃљ рЃЏрЃЮрЃ«рЃЊрЃљ:

```python
try:
    x = int("abc")
except ValueError as e:
    print("рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ:", e)
```

рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃљ:

* `ValueError` РђЊ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃбрЃўрЃърЃў
* `TypeError` РђЊ рЃерЃћрЃБрЃАрЃљрЃЉрЃљрЃЏрЃЮ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ
* `KeyError` РђЊ dict-рЃерЃўСИЇтГўтюе key
* `IndexError` РђЊ рЃАрЃўрЃљрЃерЃўСИЇтГўтюе рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў

рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃбрЃљрЃърЃќрЃћ
рЃерЃћрЃю рЃерЃћрЃФрЃџрЃЮ рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў error-handling.

1. input-рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў
