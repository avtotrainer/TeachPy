# Рюћ№ИЈ Day 4 Рєњ Concept 5

## **рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ (File Handling)**

Python-рЃерЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃЮрЃарЃЏрЃљрЃљ:

```python
file = open("data.txt", "r")
content = file.read()
file.close()
```

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ, рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃљ `with`:

```python
with open("data.txt", "r") as f:
    content = f.read()
```

**рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ:** рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃЎрЃћрЃбрЃћрЃЉрЃљ,
рЃЏрЃљрЃерЃўрЃюрЃљрЃф рЃЎрЃў, рЃЌрЃБ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ.

### рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃљ:

```python
with open("output.txt", "w") as f:
    f.write("Hello file!")
```

* `"r"` Рєњ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
* `"w"` Рєњ рЃЕрЃљрЃгрЃћрЃарЃљ (рЃњрЃљрЃЊрЃљрЃгрЃћрЃарЃА рЃФрЃЋрЃћрЃџ рЃцрЃљрЃўрЃџрЃА)
* `"a"` Рєњ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

рЃћрЃА рЃерЃћрЃюрЃЌрЃЋрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃљрЃГрЃўрЃарЃЮ рЃўрЃЦрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃфрЃљ рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃА, рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃА рЃЊрЃљ JSON рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉ.
