# ✔️ Day 6 → Concept 2

## **კონსტრუქტორი (**init**) და ატრიბუტები**

`__init__` არის სპეციალური მეთოდი,
რომელიც ავტომატურად ეშვება, როცა ობიექტს ვქმნით.

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

* `self.name` → ობიექტის ატრიბუტი
* `name` → არგუმენტი ფუნქციიდან

ობიექტის შექმნა:

```python
s = Student("Nika", 12)
```

# ✔️ Day 6 → Concept 3

## **მეთოდები (Methods) — ქცევა კლასის შიგნით**

მეთოდი არის ჩვეულებრივი ფუნქცია,
რომელიც ცხოვრობს კლასის შიგნით და მუშაობს ობიექტის მონაცემებზე.

მაგალითად:

```python
class Student:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("გამარჯობა, მე ვარ", self.name)
```

გამოძახება:

```python
s = Student("Ana")
s.greet()
```

🟦 **მნიშვნელოვანი წესი:**
ყველა მეთოდის პირველი პარამეტრი არის `self` —
ეს მიუთითებს ობიექტზე, რომელზეც მეთოდი მუშაობს.
